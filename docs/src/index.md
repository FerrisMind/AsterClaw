# AsterClaw

`AsterClaw` — легковесный AI-агент на Rust с локальным gateway, инструментами и интеграциями каналов.

## Что внутри

- Gateway c `/health` и `/ready`
- Цикл агента с tool-calls
- Конфигурируемые инструменты: `exec`, `web_search`, `web_fetch`, `fs`, `cron`, `memory`, `message`
- Канал Telegram
- Профилирование и NFR-скрипты

## Для кого эта документация

- Пользователи: запуск, конфигурация, эксплуатация
- Разработчики: архитектура, расширение инструментов, тестирование
