# Início Rápido

## Requisitos

- Rust (`stable`)
- `cargo`
- Git
- opcional: `mdbook`

## Build

```bash
cargo build --release
```

## Inicialização

```bash
asterclaw onboard
```

## Executar gateway

```bash
asterclaw gateway
```

## Verificar status

```bash
asterclaw status
```

## Visualizar docs localmente

```bash
mdbook serve docs
```
