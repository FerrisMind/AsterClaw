# Quick Start

## Requirements

- Rust (`stable`)
- `cargo`
- Git
- optional: `mdbook`

## Build

```bash
cargo build --release
```

## Initialize

```bash
asterclaw onboard
```

## Run gateway

```bash
asterclaw gateway
```

## Check status

```bash
asterclaw status
```

## Preview docs locally

```bash
mdbook serve docs
```
